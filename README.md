# myNote
​
## サイト概要
### サイトテーマ
自分専用のI関連のノート

### 主な利用シーン
通勤中や時間月越しある時にすぐにスマホを手にして見れたり、
今までまとめたものをノートで管理するのではなくこのアプリを使って一元管理できればな、、​
## 開発環境
- OS：Linux(CentOS)
- 言語：HTML,CSS,JavaScript,Ruby,SQL
- フレームワーク：Ruby on Rails
- JSライブラリ：jQuery
- IDE：Cloud9
